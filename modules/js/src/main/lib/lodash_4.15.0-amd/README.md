# lodash-amd v4.15.0

The [Lodash](https://lodash.com/) library exported as [AMD](https://github.com/amdjs/amdjs-api/wiki/AMD) modules.

Generated using [lodash-cli](https://www.npmjs.com/package/lodash-cli):
```bash
$ lodash exports=amd -o ./
$ lodash exports=amd -d -o ./main.js
```

## Installation

Using npm:
```bash
$ {sudo -H} npm i -g npm
$ npm i --save lodash-amd
```

Using an AMD loader:
```js
require({
  'packages': [
    { 'name': 'lodash', 'location': 'path/to/lodash' }
  ]
}, ['lodash/chunk'], function(chunk) {
  // use `chunk`
});
```

See the [package source](https://github.com/lodash/lodash/tree/4.15.0-amd) for more details.
